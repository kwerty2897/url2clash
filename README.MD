# Proxy URL to Clash Meta

## Supported Protocols
- `vless://` (with Reality support)
- `vmess://`
- `ss://`
- `hysteria://` (Hysteria v1)
- `hy://` (Hysteria v1)
- `hy2://` (Hysteria v2) 
- `tuic://`
- `trojan://`

## Usage

### Bash
```bash
python3 ./url2clash.py 'your_proxy_url_here'
```
*Requires: `python 3`*
